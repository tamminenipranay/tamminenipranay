Retail Analytics Project
This repository contains my retail sales analytics project, including an interactive Excel dashboard, all SQL query scripts, and detailed project documentation.

Contents
Retail_Analytics_Project_Excel.xlsx – Excel dashboard with sales, customer, and product visualizations.

Retail_Analytics_Project_Documentation.docx – Project summary, methodology, analysis, and explanations of each chart.

Retail_analytics_Project_Queries.sql – Complete set of SQL queries used for the analysis.

(optional) Dataset file, if included.

How to Use
Download and extract the zipped project folder to access all files.

Open the documentation file for a full overview of objectives, methods, and findings.

Explore the Excel dashboard for interactive data visualization.

Run the SQL queries in your preferred database software to replicate or extend the analysis.

Project Objective
Analyze retail sales data to uncover business trends, customer segmentation, top products, and sales patterns for actionable insights.
